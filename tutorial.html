Tutorial - Populus Genome Browser

DESCRIPTION

The Populus Genome Browser is developed to provide comprehensive functional annotation and visualization for individual genomes. The Populus Genome Browser leverages the genetic-molecular-phenotypic framework. It allows investigators to easily view individual genetic variants, as well as diverse genomic features and phenotypes associated to the individual genetic variants. The Populus Genome Browser can especially highlight individual genetic variants that have potential functional implications, and report the potential functional variants distribution of the input personal genome based on Populus Genome Browser.

The Populus Genome Browser is best accessed using Google Chrome, and works smoothly as well with other web-browsers, including Mozilla Firefox, Safari, Microsoft Internet Explorer (Version 10 or later), Opera, etc. HTML5 Canvas is used as the graphic engine to plot the visual elements, thus some earlier versions of web-browsers that do not support HTML5, such as IE6-9, are not compatible with the Populus Genome Browser.

The Populus Genome Browser can also be easily deployed on a user's server. A few basic supporting data/tool is required for the Populus Genome Browser local installation, such as reference sequence, cytoBand, SNP database, HGNC Genes, refSeq genes annotation, and Annovar program with SIFT and PolyPhen2 score databases. Tomcat6+ and JAVA is also needed for Populus Genome Browser running. Detailed instructions can be found in the package.

VISUALIZATION ELEMENTS OF THE FOUR TYPES OF TRACKS

Populus Genome Browser中的四种track类型显示了不同格式的基因组数据，包括序列（sequence），变异（variants），元素（elements），值（values）。序列类型只有一种显示模式，其他三种数据类型有两种显示模式：dense和pack。

1.Sequence
  在序列Sequenc Track中，参考基因组是碱基序列，由不同的颜色可视化A、C、G、T（red/yellow/blue/green）。

2.Variants
  在变异体Variants中，基因组位置和等位基因是有关遗传变异体的基本信息。在基因组面板中，带有标签的不同颜色的符号用于指示变体的位置和等位基因。与UCSC基因组浏览器类似，当浏览区域中存在大量变体时，符号会被短的垂直线（或变体覆盖较大区域的区域）替换。不同类型的变体用不同的颜色表示(SNV(blue),DEL(red),INS(blcak))。不同显示模式（dense/pcak）下，变异区域显示高度也不同。用户可以观察浏览区域的整体视图，获得变异track的可视化信息。

3.Elements
  Elements track描述了离散区域中的基因组特征。特征的属性，主要包括外显子，内含子，编码区，方向等，可视化界面使用了不同的形状来表示显示模式的差异。pack显示模式下，元素X,L,D颜色相同为blue，高度为降序排列来区分。

4.Values
  Values track描述了基因组位置/区域的特征，包括保守性，基因表达水平，DNA甲基化，组蛋白修饰和GC含量（including conservation, gene expression level, reads enrichment, DNA methylation, Histone modification, and GC content.）。Values track当前使用了波形图与热图（Waveform and Heat map）的形式可视化这些基因组数据的不同显示模式（dense/pack）。热图显示模式下正值为red，负值为blue。颜色从深到浅的程度代表了值的大小区别。

Basic Local Alignment Search Tool (BLAST)

BLAST finds regions of similarity between biological sequences. The program compares nucleotide or protein sequences to sequence databases and calculates the statistical significance of matches. BLAST can be used to infer functional and evolutionary relationships between sequences as well as help identify members of gene families.

A sequence similarity search often provides the first information about a new DNA or protein sequence. A search allows scientists to infer the function of a sequence from similar sequences. There are many ways of performing a sequence similarity search, but probably the most popular method is the “Basic Local Alignment Search Tool” (BLAST). BLAST uses heuristics to produce results quickly. It also calculates an “expect value” that estimates how many matches would have occurred at a given score by chance, which can aid a user in judging how much confidence to have in an alignment.
As the name implies, BLAST performs “local” alignments. Most proteins are modular in nature, with one or more functional domains occurring within a protein. The same domains may also occur in proteins from different species. The BLAST algorithm is tuned to find these domains or shorter stretches of sequence similarity. The local alignment approach also means that an mRNA can be aligned with a piece of genomic DNA, as is frequently required in genome assembly and analysis. If instead BLAST started out by attempting to align two sequences over their entire lengths (known as a global alignment), fewer similarities would be detected, especially with respect to domains and motifs.
